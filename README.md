# 🚀 Data Validation & Comparison Tool

A powerful Streamlit-based application for validating and comparing datasets (CSV/Excel) with Excel-style output and automated validation logic.

---

## 📌 Features

- 🔗 Column Mapping (Flexible)
- 📅 Date Formatting Support
- ⚙️ Custom Sorting per file
- 🧹 Data Normalization 
- ✅ Cell-by-cell Validation using Excel formulas
- 📊 Excel Output with structured tables
- 📥 One-click download

---

## 🧠 How It Works

1. Upload two files (CSV or Excel)
2. Map columns between File 1 and File 2
3. (Optional) Apply formatting and sorting
4. Click **Run Comparison** button
5. Download Excel output with:
   - File1 data
   - File2 data
   - Validation sheet (TRUE/FALSE using formulas)

---

## 📂 Output Structure

The generated Excel file contains:

- **File1 Sheet**
- **File2 Sheet**
- **Validation Sheet**
